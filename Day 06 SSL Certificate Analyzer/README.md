# Day 06 - SSL Certificate Analyzer

A simple Python-based tool that retrieves and analyzes SSL/TLS certificate information from a target website.

---

## Features

- Retrieve SSL certificate details
- Check certificate issuer
- Display certificate validity dates
- Calculate remaining certificate validity
- Beginner-friendly implementation

---

## Technologies Used

- Python
- SSL
- Socket Programming

---

## Installation

```bash
git clone <url>
cd Day-06-SSL-Certificate-Analyzer
```

---

## Usage

```bash
python ssl_analyzer.py
```

Enter a target domain.

Example:

```text
google.com
```

---

## Example Output

```text
Enter Domain: google.com

Certificate Information

Issuer:
Google Trust Services LLC

Valid From:
2025-01-15

Valid Until:
2025-04-15

Days Remaining:
85
```
