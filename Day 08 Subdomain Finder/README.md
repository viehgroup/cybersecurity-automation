# Day 08 — Subdomain Finder

A simple Python-based tool that discovers subdomains for a target domain using a wordlist and DNS resolution.

---

## Features

- Discover active subdomains
- DNS resolution validation
- Beginner-friendly implementation
- Lightweight and easy to use
- Useful for reconnaissance and asset discovery

---

## Technologies Used

- Python
- Socket Programming

---

## Installation

```bash
git clone https://github.com/viehgroup/cybersecurity-automation.git
cd Day-08-Subdomain-Finder
```

---

## Usage

```bash
python subdomain_finder.py
```

Enter a target domain.

Example:

```text
example.com
```

---

## Example Output

```text
Enter Domain: example.com

Scanning...

[+] Found: www.example.com
[+] Found: mail.example.com
[+] Found: api.example.com
```
